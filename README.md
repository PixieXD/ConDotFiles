## Overview

A Config DotFiles _(Hence the name "ConDotFiles")_ for my personal NixOS system
using **Hyprland & Waybar (Ooooo, waybar jumpscare......)**.

![Mafuyu](./images/showcase.png)

## Words

Back in the early days of this repo, it was structured for my Arch Linux build,
only intend to save my dotfiles in case I wanted to reinstall. _(It was a mess
back then I swear...)_ But that has since changed to a more _kind of_ clean
looking structure now that I hopped into NixOS as it fits my needs and stuff.

To be fair, never I made this far from just editing a few nix files. It's just
files I did during free time. And I had too much fun with it... Anyways, credits
time!!
