# ConDotFiles by Larry~
Hi! This seems to be in WIP (as of this commit), but if you somehow saw this git repo, welcome~!

## Overview
A Config DotFiles (Hence the name "ConDotFiles") for my personal NixOS system. (Flakes + Home Manager)

Back in the early days of this repo, it was structured for my Arch Linux build, but that has since changed to a more clean looking structure ever since I hopped into NixOS.
Now with all that out of the way, currently it is now using :

- Hyprland
- SwayOSD
- SwayNC
- Waybar
- Rofi
- Limine
- Ly
...and much more!

On NixOS side of things, Flakes and Home Manager is also being used. Although that might change as I had intentions on rewriting it into Hjem instead of HM, with flake-parts also being used (Expect this in the future ;>)
