## Overview
A Config DotFiles *(Hence the name "ConDotFiles")* for my personal NixOS system using **Hyprland & Waybar**.

![Mafuyu](./images/showcase.png)

## History & Thoughts
Back in the early days of this repo, it was structured for my Arch Linux build, only intend to save my dotfiles in case I wanted to reinstall. *(It was a mess back then I swear...)* But that has since changed to a more *kind of* clean looking structure ever since I hopped into NixOS.

Speaking of NixOS, Flakes and Home Manager is being used! Although that might change in the future as I had intentions on rewriting it into Hjem instead of Home Manager, with flake-parts if that's possible. So do expect this in the future!

Now to be fair, even though the current codebase is nice on paper, there are still lots of improvements to be made, mainly the overall configurations, it's file structure and the ability to share to other machines. **It IS NOT perfect, yet**, but if I have free time then I would definately get back to work and fix these things-
